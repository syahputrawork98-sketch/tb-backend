# TB-backend
Backend aplikasi web TB (toko bangunan) StripTB yang berfungsi untuk mengelola data, menjalankan proses bisnis, serta mendukung interaksi antara customer, customer service, dan pemilik atau admin toko. Sistem ini memungkinkan pengelolaan produk, pesanan, pengguna, dan operasional toko secara terintegrasi, aman, dan efisien.
